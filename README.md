## 📘 Day 1 — Fundamentals of Data Analysis

### 🔹 Data
Raw facts, figures, symbols, characters, or observations that can be stored and processed.

**Examples:** Letters, Numbers, Special characters, Images, Audio, Videos

### 🔹 Information
Processed and organized data presented in a meaningful context.

> Data = Raw facts → Information = Meaningful results

### 🔹 Key Concepts
- **Tables & Grids** — Data organized into rows and columns
- **Distinct Values** — Unique values after duplicates are removed
- **Big Data** — Extremely large datasets requiring advanced tools

---

### 📊 Data Analysis Approaches
1. **Descriptive** — What happened?
2. **Diagnostic** — Why did it happen?
3. **Predictive** — What is likely to happen?
4. **Prescriptive** — What should we do?

### 📐 Types of Data

| Type | Description | Examples |
|------|-------------|---------|
| **Quantitative** | Numerical, measurable | Age, Salary, Sales |
| **Qualitative** | Non-numerical, descriptive | Reviews, Gender, Feedback |

- Quantitative → Discrete / Continuous
- Qualitative → Nominal / Ordinal

---

### 💡 Key Takeaway
> *"Data is just raw facts. Through analysis, those facts become information, insights, and ultimately better decisions."*
